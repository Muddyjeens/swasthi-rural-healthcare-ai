# Swasthi Backend

FastAPI backend for the Swasthi conversational symptom assessment MVP.

## Files

- `main.py`: Compatibility entrypoint for `backend.app.main`.
- `app/main.py`: Creates the FastAPI app, enables CORS for Expo, loads the dataset classifier and the configured text model once at startup, and mounts routes.
- `app/routers/chat.py`: Exposes `POST /chat`.
- `app/routers/voice.py`: Exposes backend-ready `POST /voice-transcribe`.
- `app/routers/translate.py`: Exposes backend-ready `POST /translate`.
- `app/schemas/chat.py`: Pydantic request and response schemas.
- `app/services/symptom_classifier_service.py`: Loads the uploaded CSV, builds a dataset symptom index, loads the trained classifier artifact when available, and performs dataset-only symptom matching.
- `app/services/pretrained_healthcare_service.py`: Loads the configured text model and turns classifier context into safe patient-facing language.
- `app/services/whisper_stt_service.py`: Uses open-source Whisper speech recognition for Indian-language voice input.
- `training/train_symptom_classifier.py`: Trains Random Forest and XGBoost on the uploaded dataset, compares metrics, and saves the best model.
- `training/train_derm_foundation_classifier.py`: Uses `google/derm-foundation` embeddings from Fitzpatrick17k image URLs, then trains a skin-label classifier.
- `datasets/disease_symptoms_2023.csv`: Uploaded disease/symptom dataset used by Module 2.
- `models/symptom_classifier/`: Saved classifier artifacts generated by training.
- `requirements.txt`: Python dependencies.

## Run

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

The default chat model is `microsoft/Phi-3-mini-4k-instruct`, loaded through Transformers.
To switch models:

```bash
SWASTHI_MODEL_ID=microsoft/Phi-3-mini-4k-instruct uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

The first run downloads model weights and can take time. A GPU or Apple Silicon acceleration is strongly recommended.

Voice transcription defaults to open-source `openai/whisper-large-v3-turbo`. To use another compatible Whisper model:

```bash
SWASTHI_STT_MODEL_ID=openai/whisper-large-v3-turbo uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

## Train Symptom Classifier

```bash
backend/.venv/bin/python backend/training/train_symptom_classifier.py
```

The current full-dataset run trained Random Forest and XGBoost on 246,945 rows and 754 trainable disease groups. XGBoost won with weighted F1 `0.7326`, and the saved compressed artifact is about 2.3 MB.

## Train Derm Foundation Skin Classifier

`google/derm-foundation` is an image embedding model. Log in to Hugging Face if required, then train a classifier on the uploaded Fitzpatrick17k labels:

```bash
backend/.venv/bin/python -c "from huggingface_hub import login; login()"
backend/.venv/bin/python backend/training/train_derm_foundation_classifier.py --limit 500
```

Remove `--limit 500` for a full run. Embeddings are cached in `backend/models/derm_foundation_classifier/embeddings.joblib`.
